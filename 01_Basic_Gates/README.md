# Basic Logic Gates in Verilog

## 📌 Overview

Implementation and simulation of basic digital logic gates using Verilog HDL.  
This project introduces modular Verilog design, continuous assignments, and testbench development.

**Tools Used**
- Verilog HDL
- EDA Playground
- Icarus Verilog
- EPWave

---

## 🎯 Objectives

- Implement fundamental logic gates in Verilog.
- Practice writing synthesizable RTL code.
- Develop a Verilog testbench for functional verification.
- Analyze simulation waveforms.

---

## 🔧 Modules Implemented

| Module | Description |
|---|---|
| AND Gate | Logical AND operation |
| OR Gate | Logical OR operation |
| NOT Gate | Logical inversion |
| NAND Gate | Inverted AND operation |
| NOR Gate | Inverted OR operation |
| XOR Gate | Exclusive OR operation |
| XNOR Gate | Exclusive NOR operation |
| Tri-State Buffer | Output control using High Impedance (Z) |



## 💡 Key Learnings

- Verilog module creation
- Continuous assignment (`assign`)
- Boolean logic implementation
- Testbench writing
- Waveform analysis

---

## 🌍 Applications

Basic logic gates are the building blocks of:

- CPUs and microcontrollers
- FPGA designs
- ASIC circuits
- Digital control systems
